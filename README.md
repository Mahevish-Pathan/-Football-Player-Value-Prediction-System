# -Football-Player-Value-Prediction-System
This project focuses on predicting the market value of football players using machine learning techniques. It involves data preprocessing, exploratory data analysis (EDA), feature engineering, and model building using algorithms like Random Forest. Additionally, PCA is applied for dimensionality reduction, and GridSearchCV is used for hyperparameter tuning to improve model performance. The project provides insights into key factors influencing player value and demonstrates an end-to-end ML workflow suitable for real-world applications.
